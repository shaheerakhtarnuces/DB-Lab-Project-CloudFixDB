# CloudFixDB: A Cloud Incident & Remediation Database System ☁️🔧

## 📖 Project Overview
CloudFixDB is a relational database system designed to simulate how cloud infrastructure incidents are monitored and resolved. The system stores information about cloud resources (such as compute, storage, and network components), generated alarms, detected incidents, and the remediation actions taken to fix those issues.

## 🎯 Target Users
* **Cloud Engineers**
* **IT Support Teams**
* **DevOps Engineers**

## ✨ Key Features
1. **Resource Management:** Keep track of cloud compute, storage, and network components.
2. **Incident Tracking:** Log and monitor detected incidents in the cloud infrastructure.
3. **Alarm Management:** Manage and trigger alarms based on infrastructure anomalies.
4. **Remediation Action Management:** Record the actions taken to resolve specific incidents.
5. **Reporting & Analysis:** Generate insights and reports based on incident history and resolution times.

## 🛠️ Tech Stack
* **Database:** MySQL
* **Backend:** Python, Flask
* **Frontend:** HTML, CSS, Bootstrap
* **Additional Tools:** Excel
